# Event Organizer Client

This is the client-side application for the Event Management Platform, built with React. It provides a user-friendly interface for attendees and event organizers.

## Tech Stack

- **Framework:** React
- **Routing:** React Router
- **Styling:** Tailwind CSS
- **HTTP Client:** Axios
- **Animations:** Framer Motion, AOS
- **Charts:** Recharts
- **Icons:** React Icons
- **Build Tool:** Vite

## Features

- **Event Discovery:** Browse and search for upcoming events.
- **User Registration:** Simple and secure registration for events.
- **Dashboard & Analytics:** An intuitive dashboard for organizers to monitor event statistics like registration numbers with interactive charts.
- **Create & Manage Events:** Organizers can create, edit, and manage their events through a simple interface.

## Getting Started

To get the client application running locally, follow these steps:

1. **Navigate to the client directory:**
   ```bash
   cd client
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

This will start the development server, and you can view the application in your browser at `http://localhost:5173`.
